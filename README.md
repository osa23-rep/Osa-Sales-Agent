# Osa Sales Agent

A beginner-friendly, carrier-style recommendation web app built with Python and Flask.  
Users enter a budget, device preference (iPhone/Android), and priority (camera/battery/performance/value). The app returns a top phone recommendation, alternatives, a monthly estimate, and trade-in notes.

> This is a learning project focused on clean logic, input handling, and a simple web UI.

---

## Features

- Phone upgrade recommendations based on:
  - Budget
  - Device preference (iPhone / Android)
  - Priority (camera / battery / performance / value)
  - Trade-in toggle + condition (good / cracked / dead)
- Returns:
  - Top pick
  - 1–2 alternatives
  - Estimated monthly price (simple demo calculation)
  - Notes about trade-in and condition
- Flask web UI:
  - Form input page
  - Results page with recommendation “card”

---

## Tech Stack

- Python
- Flask
- HTML / CSS
